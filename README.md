# wxPython_WallpaperEngine

Desktop wallpaper engine based on wxPython Library

基于wxpython库实现的桌面壁纸引擎

# 技术细节

wxpython有自带的视频播放器控件，但是它每次播放完视频后会自动黑屏。为了实现无缝循环播放，我基于wx.DC尝试构建自定义的视频播放器。由于算法的更改，以前实现的功能暂时不可用。
